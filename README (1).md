# H&M Energy — Sistema de Cotizaciones

Aplicación web para generar cotizaciones y facturas de venta de **H&M Energy Solutions & Products**.


## ✨ Características

- 📋 Cotizaciones y facturas con diseño profesional
- 🗄️ Base de datos de 57 materiales/servicios **editable**
- ➕ Agregar, modificar y eliminar ítems de la BD
- 🔍 Búsqueda instantánea de ítems
- 💰 Aplicar IVA (19%) opcional
- ⬇️ Descarga en PDF de alta calidad
- 💾 Guarda automáticamente en el navegador
- 📱 Interfaz responsive

## 📱 Uso
1. Llena los datos del cliente
2. Busca y agrega ítems (los precios son editables)
3. Ve a "Vista Previa" para revisar
4. Descarga el PDF
